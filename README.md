# 🌸 Egeobutterfly

> **Proyecto Integrador - React + Vite**

## Descripción del Proyecto

**Egeobutterfly** es una Single Page Application (SPA) desarrollada con React y Vite que presenta una loción corporal de larga duración mediante una interfaz moderna, elegante y responsiva.

La aplicación fue diseñada siguiendo el paradigma de componentización de React, ofreciendo una navegación clara y una experiencia visual atractiva.

---

# Problema que Resuelve

Actualmente muchas páginas de presentación de productos poseen diseños poco intuitivos o una organización deficiente de la información, dificultando que los usuarios comprendan rápidamente los beneficios del producto.

Egeobutterfly busca solucionar este problema ofreciendo una interfaz moderna, organizada y visualmente atractiva que permite presentar de forma clara las características, beneficios y propósito de una loción corporal premium, mejorando la experiencia del usuario y facilitando la comprensión del producto.
